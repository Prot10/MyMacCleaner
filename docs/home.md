# Home & Smart Scan

The Home screen is your dashboard for quick system overview and one-click cleanup.

## Overview

When you open MyMacCleaner, the Home screen provides:
- Quick system stats at a glance
- Smart Scan for comprehensive cleanup
- Recent activity and cleanup history
- Quick action shortcuts

## Quick Stats

The dashboard displays four key metrics:

| Stat | Description |
|------|-------------|
| **Storage** | Total disk usage and available space |
| **Memory** | Current RAM usage and pressure level |
| **Junk Files** | Estimated cleanable space |
| **Apps** | Total installed applications |

## Smart Scan

Smart Scan analyzes your system across all cleanup categories in one click.

### What It Scans

1. **System Junk** - Caches, logs, temporary files
2. **User Junk** - Downloads, mail attachments
3. **Application Junk** - App caches, crash reports
4. **Development Files** - Xcode DerivedData, simulators
5. **Browser Data** - Browser caches (Safari, Chrome, Firefox)
6. **Trash** - Items in your Trash

### How to Use

1. Click the **Smart Scan** button on the Home screen
2. Wait for the scan to complete (progress shown)
3. Review the results by category
4. Click **Clean** to remove selected items

### Scan Results

After scanning, you'll see:
- Total cleanable space found
- Breakdown by category
- Individual items you can select/deselect
- Safety indicators for each item

## Quick Actions

Shortcuts to common tasks:
- **Empty Trash** - Clear your Trash instantly
- **Free Memory** - Release inactive RAM
- **View Large Files** - Jump to Space Lens

## Tips

- Run Smart Scan weekly for optimal performance
- Review results before cleaning - some cached data speeds up apps
- Use individual feature pages for more granular control
